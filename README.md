#SWAGLAB E-Commerce Application Testing

Objective: Developed an automated QA and testing solution to ensure the functionality, reliability, and performance of the Swag Labs
e-commerce application used for placing online orders.

#Tools

Selenium Webdriver
TestNg
POM
#Technical Details:

• Designed and executed automated test cases to validate end-to-end e-commerce workflows, including login, product
selection, cart operations, and checkout.
• Built automation scripts using Selenium WebDriver and TestNG for UI-based browser testing and test execution.
• Utilized Maven as a build and dependency management tool for executing automated test suites.

#Outcome of this Project

Have done the Swaglab E-commerce application testing to ensure functionality,reliability and performance for placing online orders successfully.
